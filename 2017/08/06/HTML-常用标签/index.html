HTML-常用标签

2017.08.06 Robin He 前端  热度 ℃

使用HTML标签最重要的一点就是语义化

语义化

  1. 语义化目的：
    1. 让机器（搜索引擎）易于理解，利于SEO；
    2. 阅读代码的人能够很容易理解网页各个模块并维护；
  2. 语义化姿势：
    1. 尽量使用含语义信息的标签，例如header,nav,aside,section；
    2. 标题、段落、列表等标签严格区分，不要滥用；
    3. 内容结构化，去掉css也具有清晰可读的结构；

文档标签

  1. html lang属性声明语言
  2. head
    • meta
    • title
    • link
    • style
    • base
  3. body
    • script
    • noscript

布局标签

  1. div, span
  2. main, header, footer, section, nav
  3. article, aside

表格标签

  1. table
  2. thead, tbody, tfoot
  3. caption, col, th, tr, td

表单标签

  1. form
  2. input
  3. fieldset, legend, label,
  4. textarea, select
  5. optgroup, option
  6. button
  7. datalist, meter, output, progress

列表标签

  1. ul, ol, li
  2. dl, dt, dd

图像链接音频视频

  1. img, a, figure, figcaption
  2. video, audio

内嵌标签

  1. iframe 唯一可用的框架标签
  2. embed, object, source

文本格式标签

  1. h1-h6, p
  2. br, hr
  3. strong, i
  4. pre, sup, sub, blockquote
  5. time, data, mark
